# Advanced Testing Integration - Part 2

This project implements Part 2 of the advanced software testing evaluation.

## Main objective

Develop an advanced testing integration system that includes:

- Automatic combinatorial test case generation.
- Test case prioritization by risk level.
- Learning from previous executions using a predictive model.
- A complete testing pipeline with:
  - Static analysis.
  - Unit testing.
  - Coverage testing.
  - Mutation testing.
  - Combinatorial testing.
  - No-code automation supported by AI.
  - Predictive reliability model.

- Pytest
- Pytest-Cov
